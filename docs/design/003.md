# Design Doc 003: AI+PBL Agent 架构（ReAct交互+知识模板）

**状态**: 规划中
**日期**: 2026-02-20  
**作者**: JasonXie

---

## 1. 背景与目标
对话栏交互：路由Agent不仅通过用户输入知道start from，还需要在一个环节结束后，自动导航到下一个相关环节，并通过会话栏告知用户，一直到任务完成。搞清楚目前通过graph能解决还是需要ReAct方法
知识模板：存储一些教学原则如课标和教师经验，在需要的时候调用
web_search工具的调用，在调用工具的时候对话栏会显示调用工具中...，用户可以在对话栏点击联网按钮，触发联网查找。
多方案选择：每次生成3个方案，中间栏预览显示3个方案，右侧对话栏用户选择123（每个方案的标题），通过鼠标点击选择
